# adgentiks-live-pages

**Public hosting mirror** for Adgentiks client-review pages — served via GitHub Pages at
https://jmadops.github.io/adgentiks-live-pages/

⚠️ This repo is **public and contains only finished standalone HTML** — never commit secrets,
`.env` files, or source tooling here. Pages are published automatically by
`live-pages/scripts/publish.sh` in the private `adgentiks-page-studio` repo. Don't hand-edit
here; edit the source and re-run publish.
